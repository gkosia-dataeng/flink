# Apache Flink Real-Time Data Processing Examples

This repository contains **information** and **practical examples** for processing **real-time data** using **Apache Flink**.  
It is designed to help developers learn and experiment with Flink's streaming and table APIs in realistic scenarios.


## Repository Structure

- [local_env](./local_env): Dockerized Flink-Kafka environment for local development  
- [00_Theory](./00_Theory/): Learn how Apache Flink works and its different components  
- [01_Flink_sql](./01_Flink_sql): Flink SQL examples  
- [02_Flink_Table_api](./02_Flink_Table_api): PyFlink Table API examples  
- [03_Flink_Datastream_api](./03_Flink_Datastream_api): PyFlink DataStream API examples  
- [data_producer](./data_producer): Python application that produces data to [local_env](./local_env) Kafka
